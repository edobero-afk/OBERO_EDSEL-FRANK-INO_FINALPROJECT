# PhishGuard CLI: URL Security Analyzer

## Project Description
PhishGuard CLI is a Command Line Interface application that analyzes URLs and detects possible phishing links based on common suspicious patterns such as insecure protocols, suspicious keywords, long URLs, and excessive hyphens.

This project was developed as an individual Python final project to demonstrate object-oriented programming, file handling, data structures, algorithms, and data processing.

## Features
- Analyze URLs for phishing indicators
- Display URL risk level (LOW, MEDIUM, HIGH)
- Show reasons for risk classification
- Save scan history into JSON file
- View previous scan history
- Handles invalid menu inputs
